## 开发的一个脚手架，可用于 react 和 vue 中 

### npm
```
npm i dwj-cli -g
```

### 创建 React 或 Vue 模板
```
dwj create demo -f React(or Vue)
```

### 添加组件
**默认**
```
dwj addcomponent Hello
```
**自己写路径**
```
dwj addcomponent Hello -d src/pages
```

### 后续还会添加新功能，目前还没有 Vue 的，只有 React 的
**例如：**
* 生成 router，生成 store 等等
